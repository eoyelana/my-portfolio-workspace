# eoyelana.com

Personal portfolio of Emmanuel Oyelana, Data & AI Engineer. Live at
[www.eoyelana.com](https://www.eoyelana.com).

## Stack

Next.js (App Router, TypeScript), Tailwind CSS v4, deployed on Vercel.
Project pages are data-driven: copy and code snippets live in
`src/lib/projects/` and render through shared components in
`src/components/`.

## Commands

Run everything from this directory.

```bash
npm run dev      # local dev server on http://localhost:3000
npm run build    # production build
npm run start    # serve the production build
npm run lint     # ESLint
npm run format   # Prettier
```
